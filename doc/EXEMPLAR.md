# Exemplar Project

A distributed ToDo app
* CRUD todo items
* share/link with other users for individual items or lists
* users can notate items

## Features
* administrative controls
* private data (fiinite users)
* public data

## Purpose
* Pressure testing
  * security
    * encryption
    * access controls
  * performance testing
  * offline sync
  * clock drift
* Sync
  * conflict resolution on
    * nodes
    * edges
    * all property types
* Querying
  * CRUD
  * multi-store ~ app-level cross-querying
